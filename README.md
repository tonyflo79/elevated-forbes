# Elevated Forbes — Demo Pages

Demo pages for the Forbes Wireless by Verizon partnership (operated by Elevated Wireless).

## Pages

| File | Description |
|------|-------------|
| `1-base-clone.html` | Pixel-perfect clone of Forbes membership page |
| `2-wireless-bundle.html` | 4th pricing card for Forbes Wireless bundle ($599/yr) |
| `3-order-bump.html` | Click-through overlay offering wireless as an order bump |
| `4-thank-you-upsell.html` | Post-purchase confirmation with wireless upsell |

## Usage

Open any HTML file directly in a browser. All pages are self-contained (fonts load from Forbes CDN).
